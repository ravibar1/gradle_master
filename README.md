# gradle_master
Gradle Helper Project

#Key Files
1. build.gradle
2. settings.gradle
3. Final jar will be collected in **build/libs/<YourProject.jar>**


#####Execute Tasks
 * gradle task wrapper
 * gradle task init
 * gradle task build
 
 Notes
 -----
 
 1) Custom tasks will be under Tasks--> Others --> 'Your task'
 2) You can run test tasks after build by extending test task
